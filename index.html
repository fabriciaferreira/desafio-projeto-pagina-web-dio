O que é HTML?

(Voltar)

HTML, ou HyperText Markup Language, é uma linguagem de marcação usada para estruturar páginas na web. Podemos dizer que o HTML é a estrutura do site.

Metaforando
Para entender melhor, imagine o HTML como um esqueleto: ele dá forma e sustentação à página, assim como o esqueleto é a base do corpo humano.

Como o HTML funciona?

(Voltar)

O HTML funciona através de tags, que são como etiquetas que informam ao navegador qual tipo de conteúdo está sendo exibido. Cada tag tem uma função específica, como mostrar um título, um parágravo, uma imagem ou um link. Essas tags costumam vim em pares: uma de abertura < > e uma de fechamento < / >, enquanto o conteúdo que será exibido fica entre elas. As tags também podem ter atributos, que são informações extras que modificam ou definem o comportamento daquele elemento.

Metaforando
Você pode imaginar as tags como caixinhas rotuladas. Cada uma tem um nome que define o que ela guarda (texto, imagem, link...) Os atributos seriam os adesivos colados na caixa, dando mais detalhe sobre o conteúdo.

Mas lembre-se de que há excessões, existem algumas tags que se fecham sozinhas.

Tags básicas do HTML

(Voltar)

Elementos principais da estrutura do HTML:

  1. < html >
  2. < head >
  3. < body >

Antes de começarmos a colocar o conteúdo do nosso documento, precisamos inserir a tag < html > < /html >. Dentro da tag < html >, colocamos outras duas tags: < head > < /head > e < body > < /body >. A tag < head > contém informações que não serão exibidas diretamente no conteúdo principal, mas são importantes para o navegador. Dentro dela também colocamos a tag < title > < /title > que define o título da página, ele é exibido na aba do navegador. A tag < body > é o que define o corpo do nosso documento HTML, é nessa tag que inserimos os elementos que serão vistos pelo usuário, como textos, imagens, etc.

Importante!
Temos também a declaração < !DOCTYPE html > que precisa ser inserida no começo do documento HTML, essa declaração informa ao navegador a versão do HTML utilizada na página - isso é muito importante para garantir que a página seja exibida de forma correta.

Tags de texto em HTML

(Voltar)

As tags de texto definem como o navegador irá exibir as informações de texto para o usuário. Elas são usadas para estruturar e formatar o conteúdo textual da página.

Exemplos:

  • Tag h1-h6 - define o tamanho do título
  • Tag p - define o parágrafo de um texto
  • Tag blockquote - essa tag serve para inserir uma citação longa
  • Tag u - adiciona o underline em um texto
  • Tag mark - destaca um trecho do texto
  • Tag sup - deixa o texto sobrescrito | Exemplo:sup
  • Tag sub - deixa o texto subscrito | Exemplo:sub
  • Tag i - deixa o texto em itálico
  • Tag strong - deixa o texto em negrito

Tags de fechamento automático

(Voltar)

As tags que são consideradas auto fecháveis (que não posssuem tag de fechamento), não precisam da tag de fechamento porque elas não tem conteúdo entre tags de abertura e fechamento. Elas definem elementos que são sozinhos e não precisam de conteúdo adicional dentro delas.

Exemplos:

  • < img /> - Usada para exibir imagens
  • < input /> - Usada para criar campos de entrada de dados
  • < br /> - Usada para inserir uma quebra de linha

Acessibilidade em HTML

(Voltar)

A acessibilidade em HTML é de grande importância para garantir que todas as pessoas tenham acesso às informações acessadas nos sites e aplicações web. É necessário implementar boas práticas de desenvolvimento para tornar o conteúdo fácil de ser acessado e entendido, principalmente por pessoas com deficiência. Inclusive, essas práticas melhoram a usabilidade para todos!

Exemplos:

  • alt (texto alternativo) - utilizando o atributo alt em imagens, por exemplo, você fornece descrições precisas para as imagens - esse recurso é muito útil para quem não consegue vê-las
  • < header >, < nav >, < article >, < footer > - utilizar tags como essas, facilita para os leitores de tela a entenderem a estrutura da página
